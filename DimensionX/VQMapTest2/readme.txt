Название: DimensionX World Builder 

Жанр: процедурный генератор миров 

API: C#, XNA 

Разработчики: я, один

Платформа: Win32 

Описание: 
Программа для процедурной генерации миров для приключений героя. 
Что-то вроде картогенератора из Dwarf Fortress. В создаваемых мирах прорабатывается политическая, экономическая и социальная обстановка, моделируется история, уровень технического прогресса может варьироваться от каменного века до колонии межгалактической империи, возможна магия, предусмотрена возможность моделирования столкновения цивилизаций разного уровня развития, например форпост высокотехнологичной цивилизации в классическом фэнтези-мире. 
В настоящий момент генерируемые миры могут быть 3 форм-факторов: обычный прямоугольный кусок плоской карты, мир-кольцо и планета. В планах - плоский мир из представлений древних (который на слонах, только без слонов), пузырь (внутрення поверхность сферы), затерянный мир (территория, со всех сторон окружённая непроходимыми горами) и пр. ... 
Интерфейс весь на английском, т.к. на нём проще генерировать описательные тексты для рас, обычаев, менталитета и пр.
Контент (модели, текстуры) в текущей версии весь честно потырен из других игр, в частности Mount&Blade и Civilization4, т.к. сам ни рисовать, ни моделить не умею.

Требования:
.NET Framework 4.0
XNA Framework 4.0
шейдеры 3.0

Небольшая инструкция: 
После установки, прежде чем генерировать миры,  нужно будет создать хотя бы одну базовую сетку. Для этого выберите в меню World->New..., затем кнопку "Show grids manager..." и "Добавить". Рекомендую начать с 10к Plain Square. Затем нажимаете "Build grid" и ждёте секунд 10-20. Индикатор прогресса на построение сетки прикрутить руки всё никак не доходят... :( Но построение сетки - это разовая процедура, один раз построил - и пользуешься. 
Когда сетка будет готова, форма генерации закроется, а в окне GridsManager в список добавится построенная сетка. GridsManager можно закрыть. Теперь в выпадающем списке слева от кнопки "Show grids manager..." появилась построенная сетка. По желанию можно поиграться с параметрами мира, затем нажимайте "Start generation!" 
С помощью кнопок "Save world preset" и "Load world preset" можно сохранять и загружать соответственно наборы параметров генерируемого мира. Быстрый доступ к пяти последним использованным наборам возможен через пункт главного меню World->Repeat creation from preset. 
Навигация по сгенерированному миру мышью как в Black&White. Колёсико - масштаб. Левой кнопкой хватаетесь курсором мыши за замлю под ним и тягаете куда хотите. Правой кнопкой меняете направление взгляда камеры. Двойной клик - выбрать государство, которому принадлежит локация под курсором. 
Кнопка Jump рядом с выпадающим списком государств (States) фокусирует камеру на центральной точке выбранного государства.
